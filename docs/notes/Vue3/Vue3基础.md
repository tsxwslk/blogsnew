## 1. API 风格
- 选项式API(Options API)：在vue2中使用，选项即为定义的`data`,`methods`,`watch`等等。
- 组合式API(Composition API)：vue3文档中默认的风格，处理逻辑更加清晰一些，通常会与 `<script setup>` 搭配使用。
